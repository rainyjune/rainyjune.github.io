Disable MySQL server in MAMP on Windows 10

MAMP is a great local server environment works on MacOS and Windows. You can set up a website and make it running with just a few clicks.

There are a few software packages in the app, including Apache, Nginx, PHP and MySQL. It’s an ideal solution stack for PHP developers, however, as a front end developer, I rarely use the MySQL server shipped with it. What I want to achieve is starting Apache only when I click the Start Servers button.

After spending some time googling, I realised there was no elegant way to disable MySQL server.

So I tried to inspect the MySQL server process in Task Manager:

mysqld (32 bit) is the name of MySQL server process. Now we’ve got where the mysqld.exe file is, then open File Exloper and change to the path, rename it to a different name, eg mysqld_backup.exe. Note you should stop servers before this step.

Now come back to MAMP app, click the Start Servers button again, you will find MySQL server is not running anymore.

Enjoy!
